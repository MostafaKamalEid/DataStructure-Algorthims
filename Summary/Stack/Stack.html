Definition

Stack is like an array also called LIFO (Last in First Out)

Stack is represented like a Bucket 🪣 each element to be added is inserted at the bottom of the bucket

You can access stack elements like array using random access but with the top notation

Linked Stack

  • It is a stack but with the pointer based implementation
  • What are the problems that made us shift towards the pointer based implementation?

    Sometimes u initialize a Stack for 100 elements need to use more

    Initialize Stack for 100 element only to use some of them leaving the rest unused

    Size is big problem when using the stack

To deal with this problem we will make each value point to the next value so that size depends only on how many values are given

  • Node

Each node is connected to the next node fixing the size issues as we easily add new nodes whenever is need only and no unused space stored

Code

Steps

Create a node with a certain value and now it is placed in the bottom of the stack

First node has stackTop pointing at it

Now create another node

  1. Make it point to the last node created
  1. Make the stackTop point to the new node

When deleting the last node created is removed and the stackTop is at the second to last created node

Dont move the stackTop directly

Implementation

template <class t>
class stack {
	struct node{
		t item ;  // can be any type of variable int float string etc
		node *next;	
	}
	node*top,*cur;
public:
	stack(){
		top=NULL;  // Constructor for intializing the stack with top = none
	}
	void push(t newItem){
		node *newItemPtr=new node;
		if(newItemPtr==NULL){
			cout << "Stack push cannot allocate memory";
		}
		else{
			newItemPtr->item=newItem;
			newItemPtr->next=top;
			top=newItemPtr;
		}
	}
	bool isEmpty(){
		return top==NULL;
	}
	
	void pop(){
		if(isEmpty())
			cout << "Stack empty on pop";
		}
		else{
			node*temp=top;
			top=top->next;
			temp=temp->next=NULL;
			delete temp;
		}
	void pop(t*stackTop){
		if(isEmpty())
			cout << "Stack empty on pop";
		}
		else{
			stackTop=top->item;
			node*temp=top;
			top=top->next;
			temp=temp->next=NULL;
			delete temp;
		}
	void getTop(t&stackTop){
		if(isEmpty())
			cout << "Stack empty on getTop";
		}
		else{
			stackTop=top->item;
		}
	void display(){
		cur=top;
		cout << "\nItems in the stack : ";
		cout << "["
		while (cur!=NULL){
			cout << cur->item;
			cur=cur->next;
		}
		cout << "]\n";

	}
}
