First In First Out (FIFO)

It is the complete opposite of the stack which was (First in last out)

First thing in the Queue is called Front (Begin, First, Head)

Last thing in the queue is called Rear (Back, End, Tail)

Example

The line of tickets for the cinema serves who comes first and priority is to him and last one to come will be the last served

Queue can be implemented using Array List or Linked List implementation\

Index Of Front and Rear

In our case we will make the front start at the beginning of the array while the rear will start at the MaxSize so that when we enqueue a value the rear uses remainder methods and enters the given value in index 0

Tips to keep in mind

Elements of a Queue always start from the front to the rear for example if front is 50 and rear is 70 then elements are from index (50,51,51,â€¦,70) but if the front is at 99 and the rear is at 5 for a size 100 array then elements are from index (99,0,1,2,3,4,)

If the rear is at the 0 and front is at 1 then the Queue is either full or empty and we can check from the count
